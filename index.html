GIT

  • cd*THE FILE YUO WANT*, will move you to that file, if in the directory.

  • cd .. moves back in git.

  • ls gives you a list of thigs in the directory.

  • pwd gives you the file path you are in.

  • git status checks what branch we are currently on

  • git checkout -b branch-name creates a new branch and switches to it.

  • git add -A adds the changed files in our working directory in GitHub.

  • git commit -m "" commits the changes to the repository.Keep them dicriptive and short.

  • git pull origin main pull the main branch to make sure that our current branch is
    up to date before opening a pull request in GitHub.

  • git push origin feature/add-html will send the branch you have been working on to GitHub.
